# Triple Espresso ☕

## 📌 Descripción

Este proyecto consiste en el desarrollo de la página web de una cafetería llamada **Triple Espresso**, como parte del Sprint 3.

La página incluye una sección de encabezado con información del negocio, un formulario de reservación y un pie de página con enlaces a redes sociales.

El objetivo principal fue aplicar buenas prácticas de HTML y CSS, así como el uso de Flexbox, formularios y estilos responsivos básicos.

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Metodología BEM (parcial)
- Normalize.css

---

## ✨ Funcionalidades

- Navegación interna
- Formulario de reservación con validación
- Diseño visual basado en un brief
- Efectos hover en botones y enlaces
- Uso de imágenes SVG

---

## 🚀 Mejoras futuras

- Hacer la página completamente responsiva (mobile-first)
- Agregar validaciones avanzadas con JavaScript
- Implementar animaciones más avanzadas
- Crear una sección de contacto funcional
- Optimizar el código con variables CSS
